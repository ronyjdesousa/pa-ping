NAME
paping - TCP connectivity and port availability testing utility

SYNOPSIS
paping [options] destination

DESCRIPTION
paping is an open-source utility for testing TCP/IP network connectivity between hosts.

It works similarly to ping(8), but uses TCP connections instead of ICMP echo requests.
This allows testing whether a remote host is reachable and whether a specific TCP port is accepting connections.

paping can be useful for:

 • testing service availability
 • troubleshooting firewall issues
 • measuring TCP connection response times
 • verifying remote port accessibility

A successful TCP connection generally follows the TCP three-way handshake:

 1) Client sends SYN
 2) Server replies with SYN, ACK
 3) Client sends ACK

paping measures the time required to establish the TCP connection and reports the result to the user.

OPTIONS
   -?, --help    Display help and usage information.
   -4, --4    Use IPv4 only
   -6, --6    Use IPv6 only
   -c, --count N    Stop after sending N connection attempts.
   -p, --port N    Specify the destination TCP port number.
   -t, --timeout N    Set the connection timeout in milliseconds.
          Default is 1000.
   --nocolor     Disable colored output.
